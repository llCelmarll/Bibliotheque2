# Configuration sudo pour Docker sur Synology

## Via SSH (administrateur)

```bash
# Se connecter en tant qu'admin
ssh admin@192.168.1.124

# Editer sudoers
sudo vi /etc/sudoers

# Ajouter cette ligne:
QuentinDDC ALL=(ALL) NOPASSWD: /usr/local/bin/docker

# Sauvegarder et quitter (:wq)
```

## Ou via fichier sudoers.d (recommand√©)

```bash
echo "QuentinDDC ALL=(ALL) NOPASSWD: /usr/local/bin/docker" | sudo tee /etc/sudoers.d/docker
```

Ensuite, testez avec:
```bash
ssh QuentinDDC@192.168.1.124 "sudo /usr/local/bin/docker ps"
```
